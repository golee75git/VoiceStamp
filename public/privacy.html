개인정보 처리 안내

문서 버전 1.0 · 최종 수정 2026-06-09 · Android APK, Web (Vercel)

VoiceStamp는 사진·제목·메모를 사용자 기기(또는 브라우저)에 저장합니다. 운영자가 별도 서버에 계정·사진을 모으지 않습니다.

1. 수집·이용 정보

정보목적보관
사진스탬프·PDF기기 파일 + SQLite (웹: 브라우저)
제목·메모표시·PDF·음성 입력SQLite
GPS 좌표제목에 장소 반영 (카카오 API)좌표는 DB에 저장하지 않음

2. 기기 권한

  • 카메라 — 사진 촬영
  • 마이크 — 음성으로 제목·메모 입력
  • 위치 — 제목 자동 장소 (거부 시 날짜·시간만)
  • 사진/저장소 — 갤러리 백업 (APK)

3. 제3자 제공

  • 카카오 로컬 API — GPS 좌표를 주소·건물명으로 변환 (키 설정 시)
  • OS 음성 인식 — 음성을 텍스트로 변환 (기기/OS 정책 적용)

4. 보관·파기

클라우드 동기화 없음. 앱 삭제·휴지통·브라우저 데이터 삭제로 제거할 수 있습니다.

5. 보안·이용자 권리

데이터는 기기에 평문 저장됩니다. 권한은 OS 설정에서 거부할 수 있습니다.

6. 아동

만 14세 미만을 대상으로 하지 않습니다.
