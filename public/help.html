도움말

시작하기

APK: 카메라로 촬영 → 저장 모달에서 제목·메모 입력 → 저장.

웹: 저장 목록 → 앨범에서 사진 가져오기.

스탬프 저장

  • 제목: 촬영 시각이 먼저 표시되고, 위치 권한 허용 시 구·동·건물명이 추가될 수 있습니다.
  • 저장 폴더: YYYYMMDD_장소명 형식. [선택]으로 기존 폴더 선택 가능.
  • 마이크 버튼으로 제목·메모 음성 입력 (APK).

저장 목록

  • 카드 탭 → 수정 · 길게 누르기 → 여러 개 선택
  • 선택 후 PDF 만들기·저장·공유, 이미지 저장, 휴지통 이동
  • 헤더 「설정」에서 PDF·폴더·손잡이 등 변경

Android 뒤로가기

  • 목록·설정 → 카메라 (설정은 들어온 화면으로 복귀)
  • 카메라 → 앱 종료 확인

웹 제한

  • 갤러리 앨범 백업 없음
  • 카메라·음성은 브라우저·환경에 따라 제한될 수 있음

문제 해결

  • 위치가 안 나옴 → GPS·위치 권한·카카오 API 키 확인
  • APK 키 변경 후 → android/gradlew clean 후 재빌드
