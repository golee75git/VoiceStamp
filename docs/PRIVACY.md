| 항목 | 내용 |
|------|------|

| 문서 버전 | 1.0 |

> **안내:** 이 문서는 앱 동작을 설명하는 **내부·배포용 안내**입니다. Google Play 등 스토어 등록 시 별도 정책 URL·법무 검토가 필요할 수 있습니다.

---

## 1. 개요

VoiceStamp는 사용자가 **사진과 제목·메모**를 기기에 저장·관리·PDF로 공유하는 앱입니다.  
대부분의 데이터는 **사용자 기기(또는 웹 브라우저) 안**에만 저장되며, VoiceStamp 운영자가 별도 서버에 사용자 계정·사진을 모으지 **않습니다**.

---

## 2. 수집·이용하는 정보

### 2.1 사용자가 직접 입력·생성하는 정보

| 정보 | 수집 시점 | 이용 목적 | 보관 위치 |
|------|-----------|-----------|-----------|
| 사진 | 촬영·앨범 선택 | 스탬프 저장·목록·PDF | 기기 파일 + SQLite (웹: 브라우저 저장소) |
| 제목·메모 | 저장 모달 입력·음성 입력 | 스탬프 표시·PDF | SQLite |
| PDF 파일 | PDF 보내기 | 저장·공유 | 기기 캐시/exports (웹: 브라우저) |

### 2.2 기기 권한을 통해 접근하는 정보

| 권한 | 용도 | 필수 여부 |
|------|------|-----------|
| 카메라 | 사진 촬영 | 촬영 기능 사용 시 |
| 마이크 | 음성으로 제목·메모 입력 | 음성 입력 사용 시 |
| 위치 (포그라운드) | 제목에 구·동·건물명 자동 반영 | 선택 (권한 거부 시 날짜·시간만) |
| 사진/저장소 (갤러리) | 스탬프 사진을 갤러리 VoiceStamp 앨범에 백업 | APK 저장 시 (권한 허용 시) |

### 2.3 제3자에게 전달될 수 있는 정보

| 수신처 | 전달 정보 | 목적 | 비고 |
|--------|-----------|------|------|
| **카카오 (로컬 API)** | GPS 좌표 (위·경도) | 좌표 → 주소·건물명 변환 | `EXPO_PUBLIC_KAKAO_REST_KEY` 설정 시에만 |
| **OS 음성 인식** (Google/Apple 등) | 음성 오디오 | 한국어 텍스트 변환 | `expo-speech-recognition` 경유, 기기/OS 정책 적용 |

좌표 자체는 SQLite에 저장하지 않으며, 변환된 **장소 문자열**만 제목에 포함될 수 있습니다.

---

## 3. 보관·파기

| 플랫폼 | 보관 | 파기 |
|--------|------|------|
| Android APK | 앱 내부 저장소 + (선택) 갤러리 | 앱 삭제, 휴지통·영구 삭제, 갤러리에서 직접 삭제 |
| Web | 브라우저 IndexedDB / WASM SQLite | 브라우저 사이트 데이터 삭제 |

앱에 **클라우드 동기화·원격 백업 기능은 없습니다.**

---

## 4. 보안

- 사진·메모는 **기기(또는 브라우저)에 평문**으로 저장됩니다. 기기 잠금·OS 보안에 의존합니다.
- PDF·공유 기능으로 내보낸 파일은 사용자가 선택한 앱(메신저, 메일 등)의 정책이 적용됩니다.
- 카카오 REST API 키는 git에 포함하지 않지만, APK·웹 빌드에는 **클라이언트에 포함**될 수 있습니다. [KAKAO-KEY-SECURITY.md](./KAKAO-KEY-SECURITY.md) 참고.

---

## 5. 이용자 권리

- **권한 거부:** 설정에서 카메라·마이크·위치·갤러리 권한을 끌 수 있습니다. 일부 기능만 제한됩니다.
- **데이터 삭제:** 목록·휴지통에서 삭제, 앱 제거, (웹) 브라우저 데이터 삭제.
- **문의:** 저장소 이슈 또는 배포자에게 연락 (아래 §7).
